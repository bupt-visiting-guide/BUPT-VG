# 日常生活

## 住宿

- **校内宿舍优先**：离实验室近、安全、便于认识同学。宿舍由学校统一安排，无需自行申请
- **合租注意**：若选择校外合租，与陌生人合租前明确生活习惯（作息、清洁、访客），减少摩擦

## 交通

- 到校第一天办理公共交通卡（学生折扣）
- 骑行友好城市可租赁或购买二手自行车
- 记住紧急救援电话与最近医院地址

## 饮食

| 场景 | 建议 |
| --- | --- |
| 日常三餐 | 学生食堂性价比最高，熟悉后尝试自己做饭 |
| 想换口味 | 校外小吃街与周边餐厅选择丰富，积极探索 |
| 社交聚餐 | 积极参与，这是融入本地圈子的重要机会 |

## 社交

- 加入校内兴趣社团（体育、音乐、志愿者）是认识本地学生的最自然途径
- 主动联系同期的其他访学同学，互相支持
- 主动融入课题组/班级微信群，打破访学生信息孤岛
