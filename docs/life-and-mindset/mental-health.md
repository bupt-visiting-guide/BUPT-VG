# 心态调整

## 文化冲击的四个阶段

```
蜜月期 → 沮丧期 → 适应期 → 融合期
（1–4周）  （1–3月）  （3–6月）  （6月后）
```

了解这个规律后，当你处于沮丧期时，能更平静地认识到"这是正常的，会过去的"。

## 常见心理困境与应对

### 孤独感
- 每天固定与家人视频通话（但不要超过 1 小时，避免过度依赖）
- 给自己定一个"走出宿舍"的每日最低任务，哪怕只是去图书馆坐坐

### 学业压力
- 承认语言劣势是真实的，给自己多留 20–30% 的时间完成同等任务
- 主动去 Writing Center 或 Language Support 寻求学校免费资源

### 与导师/同学的沟通障碍
- 邮件沟通比当面沟通压力小，可以反复修改再发送
- 在重要会议前用母语打草稿，再翻译，确保表达准确

## 何时寻求专业帮助

如果持续两周以上出现以下症状，请主动联系学校心理咨询中心（通常免费）：
- 失眠或过度睡眠
- 对日常活动失去兴趣
- 无法集中注意力影响正常学习

> 寻求帮助不是软弱，是成熟和自我负责的表现。
