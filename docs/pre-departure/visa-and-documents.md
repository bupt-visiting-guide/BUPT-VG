# 签证与材料

## 签证类型

根据访学目的地与时长，常见签证类型包括学生签证（F/J/Student Visa）和访客签证。建议提前在对方学校官网确认所需类型。

## 申请时间线建议

| 距出发时间 | 建议完成事项 |
| --- | --- |
| 3 个月前 | 确认对方学校邀请函、准备成绩单与在读证明 |
| 2 个月前 | 预约使领馆或签证中心，提交申请 |
| 1 个月前 | 跟进进度，准备补件材料 |
| 2 周前 | 确认签证到手，检查有效期与入境次数 |

## 常用材料清单

- [ ] 护照（有效期需覆盖访学全程 + 6 个月）
- [ ] 对方学校录取/邀请函（含日期与导师签字）
- [ ] 在读证明与成绩单（中英双语）
- [ ] 资金证明（银行流水或担保信）
- [ ] 个人陈述或研究计划（部分签证要求）
- [ ] 近期白底证件照
