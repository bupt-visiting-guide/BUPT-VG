# 行李清单

## 报到必备文件

- [ ] 学生证原件
- [ ] 访学协议或录取通知书（正式版）
- [ ] 原校开具的访学证明/介绍信
- [ ] 一寸白底证件照 4–6 张（报到、办卡等手续使用）
- [ ] 身份证原件 + 复印件

## 电子设备

- [ ] 笔记本电脑 + 充电器
- [ ] 移动硬盘或大容量 U 盘（备份重要文件）

## 生活用品

- [ ] 常用药物（感冒、肠胃、过敏等，建议备充足量）
- [ ] 中式调料小包（豆瓣酱、老干妈等，解乡愁必备）

## 当地通常可购买

衣物、洗漱用品、文具、零食等无需占用行李空间，抵达后在校内超市或网购解决即可。
