# 行李清单

## 必带物品

### 证件类
- [ ] 护照 + 护照复印件（单独存放）
- [ ] 签证原件
- [ ] 国际学生证（ISIC，享折扣）
- [ ] 驾照国际版（如有驾驶需求）

### 电子设备
- [ ] 笔记本电脑 + 充电器
- [ ] 万能转换插头（目的地插座类型）
- [ ] 移动硬盘或大容量 U 盘（备份重要文件）

### 生活用品
- [ ] 常用药物（感冒、肠胃、过敏等，当地可能难买到同款）
- [ ] 中式调料小包（豆瓣酱、老干妈等，解乡愁必备）

## 当地通常可购买

衣物、洗漱用品、文具等无需占用行李重量，抵达后在超市或网购解决即可。

## 行李重量提示

> 经济舱国际航班通常托运限重 23 kg，手提 7 kg。建议留 3–5 kg 余量给回程伴手礼。
