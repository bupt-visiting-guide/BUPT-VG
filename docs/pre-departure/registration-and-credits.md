# 报到与手续

## 报到注册流程

行政手续（接收函、访学证明、报到登记等）由双方学校老师层面协调完成，学生无需逐一办理。

### 抵校第一天 —— 你实际需要做的事

1. **行李到寝**：宿舍由学校统一安排，抵达后直接将行李带入指定宿舍即可
2. **领取校园卡**：一卡通由院系统一发放，用于食堂、图书馆、门禁、打印等。如未收到可前往一卡通服务中心（西土城/沙河均有网点）
3. **激活校园网**：校园网账号随报到材料下发，首次登录修改初始密码。宿舍有线网口开通可联系楼层管理员

## 学分转换前置手续

以下事项由双方学校教务层面协调，学生一般只需了解、无需亲自办理：

| 事项 | 负责方 |
| --- | --- |
| 确认北邮可认定的课程范围 | 原校院系教务 |
| 取得原校书面认定承诺 | 原校导师/院系 |
| 了解绩点折算规则 | 原校教务处 |
