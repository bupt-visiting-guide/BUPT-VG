# 选课策略

## 选课前的调研

1. **提前索取课程目录**：联系北邮教务处或导师，获取下学期开课列表
2. **查阅往年评价**：通过北邮本科教务系统或院系微信群向历届访学同学索取往年课评，了解难度与授课质量
3. **与导师确认**：选课方向尽量与导师研究方向贴近，便于课程内容直接转化为科研输入

## 课程搭配建议

| 课程类型 | 建议比例 | 说明 |
| --- | --- | --- |
| 专业核心课 | 40–60% | 补充北邮未开或较弱的方向 |
| 跨学科课程 | 20–30% | 拓宽视野，国内少见 |
| 实践研讨课 | 10–20% | 探索兴趣方向，结识不同专业同学 |

## 绩点与学分转换

回原校后的学分认定规则因学院而异，建议出发前与北邮教务确认：
- 最低通过分数线的折算方式
- 哪些课程可作为本科培养计划中的选修学分
